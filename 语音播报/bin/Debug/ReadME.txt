使用说明:

1.本软件有两种设置数据源格式的功能
  1-1.你可以使用Excel进行数据的导入,格式请参照目录下的"排片表.xls".
  1-2.如果你熟悉C#语言,可以参考接口文档

2.本软件所使用的语音API为百度的语音API,请访问 http://ai.baidu.com/tech/speech/tts 
  创建一个应用后,将其 API Key 与 Secret Key 填入 config 文件中的value值中
  具体请阅读目录下的"百度语音API使用说明".

3.本软件目前只支持单厅播报,如果你的排片中有多场同一时间进场,则只会播报第一场.
  以后的更新中可能会支持多厅播报.

4.本软件的使用请勿用于商业用途,如有合作,请联系软件制作者.
